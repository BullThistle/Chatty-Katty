# Installation

Install required dependencies
```
$ npm install
```

```
$ yarn install
```

```
$ npm install typescript@'>=2.1.0 <2.4.0'
```

# Build

To run in a browser
```
$ npm run electron:web
```

To build for mac
```
$ npm run electron:mac
```

To build for windows
```
$ npm run electron:windows
```
